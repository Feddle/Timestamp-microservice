Timestamp-microservice

Returns provided natural or unix timestamp as an object that contains both.

https://github.com/Feddle/Timestamp-microservice

Example usage

https://timestampt-microservice-feddle.glitch.me/December%2015,%202015

https://timestampt-microservice-feddle.glitch.me/1450137600

Example output

{ "unix": 1450137600, "natural": "December 15, 2015" }
